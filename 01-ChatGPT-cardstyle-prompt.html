Zion Coalition Page Conversion Prompt

Purpose

Use this prompt whenever you want a WordPress-exported page transformed into the clean Zion Coalition static-site style for GitHub Pages.

Copy-and-Paste Prompt

Please transform this WordPress-exported page into my clean Zion Coalition static-site style. Use this design: - outer background: #9fc7db - main centered page: #5fa3c7 - content cards: #d6eaf8 - rounded corners - soft shadows - dark blue headings - clean readable typography - remove WordPress admin/theme clutter - preserve the page’s actual content - convert the content into logical card sections - use relative links like index.html where appropriate - give me the finished standalone HTML file I can paste into GitHub

File Naming Rules

  • Use lowercase filenames.
  • Use hyphens instead of spaces.
  • End each page with .html.
  • Example: possession-tv.html

Image Rules

  • Put images in the images folder.
  • Use simple image names like jesus-pillow.png.
  • Reference images like this: images/jesus-pillow.png.
